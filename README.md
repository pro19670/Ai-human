# AI휴먼 - 행정·인증 대행 플랫폼

## 프로젝트 개요

**AI휴먼**은 기업의 각종 행정·인증 등록/연장 대행을 전문으로 하는 플랫폼입니다. 연구소 신설부터 나라장터 등록까지, 복잡한 행정 절차를 전문가가 원스톱으로 처리합니다.

### 주요 특징
- **다크톤 프리미엄 디자인**: 신뢰감과 전문성을 강조하는 세련된 UI
- **AI 챗봇 상담**: 로그인 사용자 대상 실시간 상담 및 견적 안내
- **회원가입/로그인**: PBKDF2 기반 보안 인증 시스템
- **무패키지 구현**: Node.js 내장 모듈만 사용한 경량 구현
- **반응형 디자인**: 모든 디바이스에서 최적화된 사용자 경험

## 🛠 기술 스택

### 백엔드
- **Node.js**: 내장 모듈만 사용 (http, fs, path, crypto)
- **인증**: PBKDF2 기반 비밀번호 해싱, HttpOnly 쿠키 세션
- **데이터 저장**: JSONL 파일 기반 (users, sessions, leads)
- **보안**: Rate limiting, CSRF 토큰, 입력 검증

### 프론트엔드
- **HTML5**: 시맨틱 마크업, 접근성 최적화
- **CSS3**: 순수 CSS, CSS Grid/Flexbox, 다크톤 디자인
- **JavaScript**: ES6+, 모듈화, 비동기 처리

### 디자인
- **컬러**: 다크 배경 (#0B0F14) + 민트 포인트 (#5EEAD4)
- **타이포**: system-ui, Inter, Pretendard 폰트 스택
- **컴포넌트**: 카드형 디자인, 14-16px 라운드, 마이크로 인터랙션

## 🚀 설치 및 실행

### 필수 요구사항
- Node.js 16.0.0 이상
- PM2 (선택사항, 권장)

### 로컬 개발 환경 설정

1. **저장소 클론**
   ```bash
   git clone <repository-url>
   cd webapp
   ```

2. **포트 정리** (선택사항)
   ```bash
   npm run clean-port
   ```

3. **서버 실행**
   ```bash
   # 직접 실행
   npm start
   
   # 또는 PM2 사용 (권장)
   pm2 start ecosystem.config.cjs
   ```

4. **접속 확인**
   ```bash
   npm run test
   # 또는 브라우저에서 http://localhost:3000
   ```

### PM2 관리 명령어
```bash
# 서비스 상태 확인
pm2 list

# 로그 확인 (논블로킹)
pm2 logs aihuman-webapp --nostream

# 서비스 재시작
npm run clean-port && pm2 restart aihuman-webapp

# 서비스 중지
pm2 stop aihuman-webapp

# 서비스 삭제
pm2 delete aihuman-webapp
```

## 🔒 보안 및 개인정보

### 보안 기능
- **비밀번호 암호화**: PBKDF2 (100,000회 반복)
- **세션 관리**: HttpOnly 쿠키, 24시간 만료
- **Rate Limiting**: IP별 분당 60회 제한
- **CSRF 보호**: 토큰 기반 요청 검증
- **입력 검증**: 모든 사용자 입력 검증 및 이스케이프

### 개인정보 처리
- **수집 목적**: 상담 서비스 제공
- **보관 기간**: 서비스 완료 후 관련 법령에 따라 처리
- **암호화 저장**: 모든 개인정보 암호화 보관
- **접근 제한**: 권한 있는 담당자만 접근 가능

## 🤖 AI 챗봇

### 지원 인텐트
- **service_info**: 특정 서비스 설명
- **pricing_scope**: 가격 범주 안내
- **document_check**: 필요 서류 안내
- **timeline**: 처리 기간 안내
- **lead_capture**: 상담 신청 유도
- **handoff**: 전문가 연결

### 챗봇 기능
- 규칙 기반 응답 시스템
- 서비스별 맞춤 안내
- 상담 예약 연결
- 히스토리 관리 (20개 제한)
